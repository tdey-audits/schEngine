## Question 1

The 4th term of an AP is 14 and the 12th term is 70. What is the first term $a$?

**Answer:** (A)

**Solution:**
1. Use the formula $a_n = a + (n-1)d$
2. Set up two equations: $a + 3d = 14$ and $a + 11d = 70$
3. Subtract the equations to find $d = 7$
4. Substitute $d$ to find $a$

**Derivation:**
```
a + 3d = 14 \\ a + 11d = 70 \\ (a + 11d) - (a + 3d) = 70 - 14 \\ 8d = 56 \implies d = 7 \\ a + 3(7) = 14 \implies a = 14 - 21 = -7
```

Reason (R): The common difference $d$ of an AP is given by the difference between consecutive terms $a_n - a_{n-1}$.

**Answer:** (A)

**Solution:**

3. Calculate the common difference $d = a_2 - a_1$.
4. Observe that the coefficient of $n$ in a linear expression of $n$ represents the common difference.

**Derivation:**
```
$a_1 = 3 + 4(1) = 7 \\ a_2 = 3 + 4(2) = 11 \\ d = a_2 - a_1 = 11 - 7 = 4 \\ \text{Since } d = 4, \text{ Assertion is true.} \\ \text{The formula } d = a_n - a_{n-1} \text{ is the standard definition, so Reason is true and is the correct explanation.}
```

## Question 13

Assertion (A): The sum of the first $n$ terms of an arithmetic progression is given by $S_n = \frac{n}{2}[2a + (n-1)d]$, where $a$ is the first term and $d$ is the common difference.
Reason (R): The sum of the first $n$ terms of an AP can be calculated as $S_n = \frac{n}{2}(\text{first term} + \text{last term})$.

**Answer:** (B)

**Solution:**
1. The formula $S_n = \frac{n}{2}[2a + (n-1)d]$ is the standard formula for the sum of $n$ terms of an AP, making Assertion (A) true.
2. The formula $S_n = \frac{n}{2}(a + l)$ where $l$ is the last term ($a_n$) is also a valid and standard formula for the sum of $n$ terms, making Reason (R) true.
3. While both are true, Reason (R) is a different formulaic representation and does not act as the logical explanation for the derivation of the formula provided in the Assertion.

**Derivation:**
```
S_n = \frac{n}{2}(a + a_n) \implies S_n = \frac{n}{2}(a + a + (n-1)d) = \frac{n}{2}(2a + (n-1)d)
```

## Question 14

Assertion (A): The sum of the first $n$ terms of an arithmetic progression is given by $S_n = 3n^2 + 5n$. The common difference of this AP is 6.
Reason (R): For an AP where $S_n = an^2 + bn$, the common difference is always $2a$.

**Answer:** (A)

**Solution:**
1. To find the common difference from the sum of $n$ terms, calculate $S_1$ and $S_2$.
2. $S_1 = a_1 = 3(1)^2 + 5(1) = 8$.
3. $S_2 = a_1 + a_2 = 3(2)^2 + 5(2) = 12 + 10 = 22$.
4. Calculate $a_2 = S_2 - S_1 = 22 - 8 = 14$.
5. Common difference $d = a_2 - a_1 = 14 - 8 = 6$. Assertion is true.
6. For a general sum $S_n = An^2 + Bn$, $a_n = S_n - S_{n-1} = A(n^2 - (n-1)^2) + B(n - (n-1)) = A(2n-1) + B = 2An - A + B$. Since $a_n$ is a linear function of $n$, the common difference is the coefficient of $n$, which is $2A$. Reason is true and correctly explains the assertion.

**Derivation:**
```
$S_n = 3n^2 + 5n \\ a_n = S_n - S_{n-1} \\ a_n = 3n^2 + 5n - [3(n-1)^2 + 5(n-1)] \\ a_n = 3n^2 + 5n - [3(n^2 - 2n + 1) + 5n - 5] \\ a_n = 3n^2 + 5n - 3n^2 + 6n - 3 - 5n + 5 \\ a_n = 6n + 2 \\ d = a_n - a_{n-1} = (6n + 2) - (6(n-1) + 2) = 6$
```

## Question 26

The sum of the first $n$ terms of an Arithmetic Progression is given by $S_n = 3n^2 + 5n$. Find the $n^{th}$ term of this AP and hence find the $20^{th}$ term.

**Answer:** $6n+2$; $122$

**Solution:**
1. Use the relation $a_n = S_n - S_{n-1}$ to find the general term.
2. Substitute $S_n = 3n^2 + 5n$.
3. Calculate $S_{n-1} = 3(n-1)^2 + 5(n-1)$.
4. Subtract $S_{n-1}$ from $S_n$ and simplify the expression.
5. Substitute $n=20$ into the resulting formula to find $a_{20}$.

**Derivation:**
```
a_n = S_n - S_{n-1} \\ a_n = (3n^2 + 5n) - [3(n-1)^2 + 5(n-1)] \\ a_n = 3n^2 + 5n - [3(n^2 - 2n + 1) + 5n - 5] \\ a_n = 3n^2 + 5n - [3n^2 - 6n + 3 + 5n - 5] \\ a_n = 3n^2 + 5n - 3n^2 + n + 2 = 6n + 2 \\ a_{20} = 6(20) + 2 = 122
```
